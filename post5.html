Return Types

Return types are commonly used with functions to denote the type of value that is supposed to be returned. Various types of returns are the standard data types such as:
Integers
Double
Long
Char
String
Objects...
or Void to denote no return type

Using return types helps ensure the the data is manipulated in a way that is expected and is more of an integrity and accuracy measure in programming since functions can have inputs that are not of the return type of the function.
