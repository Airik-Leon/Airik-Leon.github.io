Casting types

Casting types are a means to manipulate variables for different cases that may need them. Like casting a (double) to an (int) for a truncated whole number or an int as a double for a decimal place.

Casting works with number values and allows for manipulations to occur in cases like a function where the input is a different type than the return type.

With that said you can't cast integers to strings as you have to convert data type to get the desired effect.
