Loops

NOTE: All loops can achieve the same thing. Some just need more work than others.

For

For Loops are Loops that are based on a condition and iterate for a specified amount of time, Hence "For" as in For x amount of times. The notation for 'For Loops' is

for(x=0; x<10< x++){
"Do something"
}
this will stop the code at 10 since it will not add if x is 10 or greater.


While

A while is another condition based loop that accepts a condition and repeats itself until the condition is no longer met, much like the for loop. The difference is that in a while loop you need to (but not always) update the condition so the loop will terminate once the condition is no longer met. An example is:
x= 0;
While (x < 10){
.....do something;
x++;
}

do while

The do while loop is probably the most different loop because it executes at least once. It does this by automatically running the code and then checking if the condition is still in bounds.
Example

x=0; do{
....code...
x++;
}while(x < 10);
The one noticeable difference is that 'do' comes first and the 'while' is outside of the brackets along with the semi-colon.

As previously mentioned you can accomplish all of the tasks you need with any of the three loops but it is best to choose the loop that best fits your scenario.
